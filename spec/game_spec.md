# 3Dシューティングリズムゲーム 仕様書

## 1. ゲーム概要
ネオン・サイバーパンクの電脳空間を舞台にした、完全3D空間浮遊式のシューティングリズムゲーム。
画面奥からBGM（テクノ・EDM）のビートに合わせて浮遊しながら迫ってくるアルファベットのノーツに対し、マウスでエイム（照準）を合わせ、タイミングよくそのアルファベットキーを入力してレーザーで破壊する、超爽快なゲーム体験を提供する。

## 2. コアメカニクス（基本操作とルール）
* **ビジュアル・視点**:
  * **ネオン・サイバーパンクテーマ**: 暗闇の中に輝くシアン、マゼンタ、グリーンの鮮烈なネオンカラー。
  * **完全3D空間浮遊式**: プレイヤーは電脳空間を浮遊しているかのような感覚。背景には遠くへ流れるネオングリッドや星くず、デジタルノイズが走る。
  * カメラはBGMの重低音（キック）に合わせて僅かに振動（ビートパンプ）し、臨場感を高める。
* **エイム（照準）**:
  * マウス操作により、画面上を自由に動くサイバーレティクル（照準）を動かす。
  * レティクルがノーツに重なると、ロックオン演出（ネオンの枠線が縮まる、リンクラインが繋がるなど）が発生。
* **ノーツ（ターゲット）**:
  * 3D空間の奥から手前（カメラ方向）に向かって、リズム（BPM）に同期して浮遊しながら迫ってくる。
  * ノーツは立体的なキューブや球体で、表面に輝くアルファベット（`A`〜`Z`）が表示されている。
* **破壊アクション**:
  1. マウスでノーツにエイムを合わせる（ロックオン状態にする）。
  2. ノーツが画面手前の「判定リング（ジャストタイミングの距離）」に重なった瞬間に、そのノーツのアルファベットキーを押す。
  3. 正しいタイミングとキー入力で、レティクルから激しいネオンレーザーが発射され、ノーツが電子の火花（パーティクル）を散らして爆発四散する。
* **判定・スコア**:
  * 距離/タイミングに応じて `PERFECT` (虹色ネオン) / `GREAT` (シアン) / `GOOD` (イエロー) / `MISS` (レッド) の評価。
  * コンボシステム：連続で破壊成功するとコンボ数が上昇し、スコア倍率アップ＆背景のネオンがより激しく輝く（フィーバー状態）。
  * ライフ（HP）：画面端までノーツを撃ち漏らして（防ぎきれずに通過させて）しまった際（MISS）に減少。空撃ち（エイムが合っていない状態での打鍵やキーミス）ではHPは減少しない（コンボは途切れる）ため、ペナルティを過度に恐れることなくリズムに乗って楽しく連射できる。HPゼロでゲームオーバー（シャットダウン演出）。

## 3. 音楽とリズムの同期システム
* **ノリノリのテクノ・EDM BGM**:
  * ゲームに搭載された、BPM（テンポ）がはっきりしたオリジナルテクノ/EDM楽曲（Web Audio APIによるシンセ演奏、またはループ音声）。
  * ノーツはBGMのBPM（例: 128〜140 BPM）と完全に同期して、1拍、半拍などのタイミングで自動生成される。
  * プレイヤーがキーを叩いた音（SE）自体も、テクノのドラムやシンセのサンプラー音（「シャン！」「ピキーン！」など）になっており、叩くだけでセッションしているような気持ちよさを味わえる。

## 4. 追加の演出・アシスト機能
* **フィーバーモード**:
  * コンボが20回以上繋がると「フィーバー状態」に突入！
  * 背景のネオンラインがさらに激しく明滅し、BGMのビート感が増してノリノリになる。
  * フィーバー中は獲得スコアが2倍になる。
* **レーザーカラー変化**:
  * タイミングや入力キー（あるいはノーツの種類）によって、発射されるレーザーの色が鮮やかなシアン、マゼンタ、グリーンなどに変化し、画面がド派手に彩られる。
* **アシストライン（予想軌道）**:
  * 画面奥から手前に飛んでくるノーツの予想軌道がうっすらとネオンラインでガイド表示されるため、エイムが合わせやすく初心者でも安心。
* **エイム最適化とロックオン救済システム (NEW!)**:
  * **動的3Dスケール判定**: ロックオン判定の当たり判定（半径）を、ノーツの3D透視縮尺（`scale`）に同期。ノーツが近づいて巨大化するほど、直感的にエイムを合わせやすいよう判定エリアがダイナミックに拡大する。
  * **手前優先フィルタリング**: まだ奥にいる遠すぎるノーツ（Z > 1150）にエイムが奪われるのを防ぎ、手前の今すぐ叩くべきターゲットを正確に捉えられる。
  * **手ブレキーアシスト（救済判定）**: プレイヤーがキーを押したその瞬間、わずかな手ブレやフレームの揺らぎでカーソルがノーツの中心からズレてロックオンが一瞬外れてしまっても、カーソル周辺の有効エリア内に一致するアルファベットキーのノーツがあれば、自動で引き寄せて破壊（ヒット）扱いにする高精度救済システムを完備。
